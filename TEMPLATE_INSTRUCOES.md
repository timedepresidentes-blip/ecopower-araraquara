# EcoPower — Template Base de Site para Franqueados

## Como usar este template

Este é o site base da EcoPower Araraquara, pronto para ser adaptado para outras unidades franqueadas.

## O que deve ser alterado para cada nova unidade:

### Dados da unidade
- Nome da cidade (buscar e substituir "Araraquara" pelo nome da nova cidade)
- Estado (buscar e substituir "SP" se necessário)
- Endereço completo na seção de mapa
- Telefone e WhatsApp (número no botão flutuante e links href)
- Email de contato (flavio.araraquara@ecopowerfranquia.com.br)
- Link do Google Maps embed na seção id="ecopower-mapa"

### Imagens
- Foto do balão EcoPower (balao-ecopower.jpg)
- Fotos da galeria de projetos realizados
- Vídeo de fundo do hero (hero-bg.mp4)
- Fotos da frota local

### Analytics e Formulário
- ID do Google Analytics (G-YHT95RDK2X) — substituir pelo ID da nova unidade
- Código do Formspree (xzdoqpqz) — substituir pelo código da nova unidade

### Vídeos
- Atualizar os IDs dos vídeos do YouTube na seção id="ecopower-videos"

## Estrutura de arquivos
- index.html — arquivo principal do site
- hero-bg.mp4 — vídeo de fundo do cabeçalho
- balao-ecopower.jpg — foto do balão EcoPower
- TEMPLATE_INSTRUCOES.md — este arquivo
